贡献指南

关于如何贡献

以下将介绍如果提交自己的经验帖文件，过程中若存在问题，可以联系维护者 a2194037898@163.com 咨询解决。感谢大家对 JNU BAOYAN 的鼎力支持！💖

第 1 步：经验帖撰写

本项目中经验帖文件的渲染、部署均基于 Markdown 文件格式，因此我们希望大家在撰写经验帖的过程中可以选择该格式文件。
当然考虑到每个人都有自己的笔记习惯，我们也接受 Word 或 其他格式的经验帖文件，提交后将由维护人员将其转为 Markdown 文件并上线网站。

基于 Markdown 格式

Markdown 是一种轻量级、高效的文件格式，其支持多种文本格式、图片、链接、表格、数学公式等内容。该文件的后缀名为 .md。

对于 Markdown 经验帖的格式，可以参照网站中他人经验帖的格式，或使用我们所提供的模板文件进行撰写。

  • 对于熟悉 Markdown 的同学，可以使用 VsCode、Typora 等文本编辑器进行撰写。

  • 对于初次使用的同学，可以考虑网页版的 在线 Markdown 编辑器，进入网页后点击 “开始写作” ，完成后点击 “保存内容” 并将得到的 Markdown 文件重命名。

文件命名要求：学院 + 专业 + 昵称（可以是本名，也可以是你使用的网名）+ 夏令营/预推免 经验帖

示例：计算机学院 + 计算机 + Ranze + 夏令营经验帖

基于其他格式

如果您不想使用 Markdown 来编写经验帖，本站也接受 Word 和其他格式的经验帖文件，在编写完成之后，将文件重命名为：学院 + 专业 + 昵称（可以是本名，也可以是你使用的网名）+ 夏令营/预推免 经验帖

示例：计算机学院 + 计算机 + Ranze + 夏令营经验帖

第 2 步：选择封面图片

如您所见，在每一个经验帖的开头都有一张封面图片，您可以根据自己的喜好，选择一张自己喜欢的图片作为封面，如果未指明想要的封面图片，我们将从网上随机挑选一张风景照片作为封面图片。

第 3 步：文件提交

在完成经验帖的内容撰写及封面照片选择后，将其打包到一个文件夹，命名格式为：学院 + 专业 + 昵称（可以是本名，也可以是你使用的网名）+ 夏令营/预推免 经验帖

随后将文件夹打包压缩成 zip 或 rar 等格式，并发送到邮箱： 2194037898@qq.com，邮件标题同压缩包文件名。

我们将对经验帖和封面图片进行审核，确保其中不包含违法、低俗、负能量等内容。确认无误后将上线网站。
